    - `inicio.css`: Estilos CSS para la página de inicio.
  - **Nike**: Contiene archivos relacionados con la sección de Nike.
    - `nike.html`: Página principal de la sección de Nike.
    - - `inicio.css`: Estilos CSS para la página de inicio.
  - **Adidas**: Contiene archivos relacionados con la sección de Adidas.
    - `adidas.html`: Página principal de la sección de Adidas.
    - - `inicio.css`: Estilos CSS para la página de inicio.
  - **Newbalance**: Contiene archivos relacionados con la sección de New Balance.
    - `newbalance.html`: Página principal de la sección de New Balance.
    - - `inicio.css`: Estilos CSS para la página de inicio.
  - **Crocs**: Contiene archivos relacionados con la sección de Crocs.
    - `crocs.html`: Página principal de la sección de Crocs. 
    - - `inicio.css`: Estilos CSS para la página de inicio.

# Descripción del Proyecto

Este repositorio contiene el código CSS para una página web. A continuación, se detallan algunos aspectos importantes del código.

## Estructura de Carpetas y Archivos

- Todos los archivos CSS se encuentran en la raíz del repositorio.
- Las imágenes utilizadas están almacenadas en la carpeta `/historia/Imagenes/`.

## Tecnologías Utilizadas

- HTML
- CSS

## Características y Funcionalidades Principales

1. **Modelo de Caja y Fuente:**
    - Se establece el modelo de caja para incluir el borde en las dimensiones de los elementos.
    - La fuente predeterminada para todos los elementos es 'Times New Roman' o una fuente similar.
    - Aplicación de desplazamiento suave al desplazarse por la página.

2. **Estilos del Encabezado:**
    - Fondo y estilos para el logotipo y la barra de navegación.

3. **Estilos de Secciones:**
    - Estilos específicos para secciones como historia (`#paragrafo`), Streetware 1 (`#paragrafo1`), Streetware 2 (`#paragrafo2`), Streetware 3 (`#paragrafo3`), y Otros (`#otros`).

4. **Estilos de Cartas:**
    - Estilos aplicados a elementos con la clase `.carta` dentro de la sección Otros.

5. **Estilos de Pie de Página:**
    - Colores y disposición específicos para el pie de página.
